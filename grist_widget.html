Dashboard Quản Lý Dự Án
Widget này đọc dữ liệu trực tiếp từ bảng Grist của bạn. Cần cấu hình tên cột một lần duy nhất.

Cách thêm widget này vào Grist:
1. Trong trang Grist → click Add New → Add Widget to Page
2. Chọn Custom → mục Custom URL
3. Dán URL của file HTML này vào
4. Chọn bảng dữ liệu cần hiển thị
5. Nhấn nút bên dưới để cấu hình tên cột

⚙️ Cấu hình tên cột
Điền đúng tên cột trong bảng Grist của bạn (phân biệt hoa/thường).
Ví dụ cột tên dự án trong Grist tên là Project_Name thì điền Project_Name.
